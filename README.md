# Mathnote

Mathnote is a small Express + static frontend application for step-by-step math solving, graph analysis, saved notes, animation playback, and OCR-based problem capture.

## Setup

1. Copy `.env.example` to `.env`.
2. Set `GROQ_API` in `.env`.
3. Install dependencies with `npm install`.
4. Start the app with `npm start`.

The app runs at `http://localhost:3000/mathnote` by default.
